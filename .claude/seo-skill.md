# SEO Skill — claude-seo

Repo: https://github.com/AgriciDaniel/claude-seo

## Instalación (solo en Claude Code CLI terminal, no en extensión IDE)

```bash
git clone --depth 1 https://github.com/AgriciDaniel/claude-seo.git
bash claude-seo/install.sh
```

## Comandos clave para este proyecto

| Comando | Uso |
|---|---|
| `/seo audit https://www.marthacdistribuciones.com` | Auditoría completa (LCP, CLS, schema, contenido) |
| `/seo technical <url>` | SEO técnico (robots, sitemap, canonicals, redirects) |
| `/seo schema <url>` | Validar/generar schema.org |
| `/seo ecommerce <url>` | Schema de productos y e-commerce |
| `/seo images <url>` | Auditoría de imágenes |
| `/seo google setup` | Conectar Search Console + PageSpeed API |

Genera `FULL-AUDIT-REPORT.md` y `ACTION-PLAN.md` en la raíz del proyecto.

## Nota

`/plugin marketplace add` solo funciona en Claude Code CLI (terminal), no en la extensión de VSCode/JetBrains.
