# MINIMALIST SKILL — Premium Utilitarian Design

## Activation
- Command: `/minimalist`
- Phrases: "clean design", "editorial style", "minimal UI", "less is more"

## Typography Foundation
- Primary sans-serif (body, UI, buttons): Geist Sans, SF Pro Display, or system-native with character
- Serif for editorial headings: Lyon Text or similar — tight tracking, minimal line-height
- BANNED: Inter, Roboto, generic system fonts

## Color Strategy
- Base palette: warm monochrome neutrals — #FFFFFF, #F7F6F3, #111111
- Accents: muted pastels for semantic signals only — one color, used sparingly
- BANNED: vibrant primaries, gradients, neons, AI purple/blue aesthetic

## Layout Principles
- Asymmetric bento-grid preferred over symmetric 3-col
- Border-radius: crisp 8-12px — not pill, not sharp
- Borders: 1px only, no heavy shadows
- Generous internal padding — breathing room is the design
- Content width constrained — never full-bleed text
- Establish macro-whitespace FIRST, then add content

## Component Defaults
- Primary buttons: solid charcoal (#111111), no shadow, hover = subtle scale
- No heavy drop shadows anywhere
- No standard icon libraries (Lucide, Feather) — use Phosphor or custom
- No emoji in any context
- Ambient imagery for depth over flat colored backgrounds

## Motion (invisible by design)
- Fade-in entries — nothing dramatic
- Ultra-diffuse shadow shifts on hover
- Staggered list reveals with minimal delay
- Animate ONLY `transform` and `opacity`
- No bounce, no spring physics — smooth ease-out

## BANNED
- Inter, Roboto
- Lucide, Feather icons
- Heavy drop shadows
- Emoji
- "Seamless", "Unleash", "Next-Gen" copywriting
- Vibrant gradients
- Centered hero with big H1 (DESIGN_VARIANCE default: high)
